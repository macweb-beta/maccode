In the <video> tags, we added <source> tags with the src attribute to link and display the video. Two <source> tags are used because the browser may not recognize the first file type (the type attribute), but it can display the next recognized source. If none is recognized, the browser displays the text within the <video> tags. The <video> tags have the controls attribute for video control. <audio> tags are the same, but the width attribute cannot be used for styling. We already learned about <img> tag here.
